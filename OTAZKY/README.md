# OTÁZKY METODIKY

Tento adresár obsahuje definície otázok. Neobsahuje konkrétne odpovede, dôkazy, Validácie ani Autority Validácií.

Otázky sú rozdelené podľa úrovne použitia:

```text
UNIVERZALNE/  = otázky použiteľné na preverovanie čohokoľvek
ATRIBUTOVE/   = otázky odvodené pre konkrétny preverovaný atribút
PROJEKTOVE/   = otázky viazané na konkrétny projektový predmet
```

Konkrétne výsledky použitia otázok patria do samostatného priestoru `HODNOTENIA/` a neskôr do príslušných databázových záznamov.
