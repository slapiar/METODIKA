# ATRIBÚTOVÉ OTÁZKY

Tento adresár obsahuje otázky odvodené pre konkrétne preverované atribúty.

Atribútová otázka:

- vychádza z univerzálnej principiálnej otázky,
- je viazaná na presne určený atribút,
- ešte nemusí byť viazaná na jeden konkrétny projektový predmet,
- tvorí základ pre konkrétne hodnotenie atribútu vo veci.

Atribúty sa ukladajú podľa svojho významového zaradenia. Disciplína je preto uložená pod Zodpovednosťou.
