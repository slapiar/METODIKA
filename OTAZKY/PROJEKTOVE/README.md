# PROJEKTOVÉ OTÁZKY

Tento adresár obsahuje otázky viazané na konkrétnu vec, jav, úlohu, rozhodnutie, osobu, proces alebo projektový predmet.

Projektová otázka:

- vychádza z univerzálnej alebo atribútovej otázky,
- presne určuje predmet hodnotenia,
- určuje potrebný vzťah, rozsah a čas,
- musí umožniť vytvorenie jednoznačného hodnotiaceho záznamu.

Samotné odpovede, dôkazy, Validácie a Autority Validácií sem nepatria.
