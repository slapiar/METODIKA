# UNIVERZÁLNE OTÁZKY

Tento adresár obsahuje otázky použiteľné na preverovanie ľubovoľného javu, predmetu, procesu, rozhodnutia alebo skutku.

Univerzálna otázka:

- nie je viazaná na konkrétny atribút,
- nie je viazaná na konkrétny projekt,
- určuje všeobecnú principiálnu pozíciu skúmania,
- môže byť základom atribútovej alebo projektovej otázky.

Aktuálny balík:

- `Objektivita-XY.md` — univerzálna objektívna matica 49 otázok.
